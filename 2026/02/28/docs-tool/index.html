常见有这些（python-docx 语义）：

  • doc.paragraphs：正文顶层段落集合
  • doc.tables：正文顶层表格集合
  • doc.sections：分节集合（页边距、页眉页脚等版式）
  • doc.styles：样式集合（段落/字符/表格样式定义）
  • doc.inline_shapes：内联图形集合（图片等）
  • doc.core_properties：文档元数据（标题、作者、创建时间等）
  • doc.settings：文档设置（部分全局开关）
  • doc.element：底层 XML 根节点（可直接操作 w:p/w:r/w:t 等）
  • doc.part：OPC 部件入口（关系、部件级操作）
