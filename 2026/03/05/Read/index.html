A Silicon Surface Code Architecture Resilient Against Leakage Errors

目前，超过表面码等纠错码阈值的单量子比特高保真度门操作已得到验证，双量子比特门保真度也达到98%且正在快速提升。然而，量子点阵列中还存在其他类型的错误，例如电荷泄漏和传播，这些错误无法通过量子纠错码进行纠正，因此即使发生概率很低也可能造成严重破坏。我们提出了一种基于硅量子点自旋量子比特的表面码架构，通过引入多电子中介点来增强对泄漏错误的鲁棒性
